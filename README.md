# 📘 Baekjoon Java 문제풀이 정리

이 저장소는 백준(BOJ) 알고리즘 문제를 Java로 풀이한 코드들을 정리한 공간입니다.  
문제는 **난이도(티어)별 디렉토리**로 정리되어 있으며,  
각 파일은 **문제 번호 + 문제 이름** 형태로 구성되어 있습니다.

---

## 📁 폴더 구조

| 폴더 | 설명 |
|------|------|
| `bronze/` | 브론즈 문제 풀이 |
| `silver/` | 실버 문제 풀이 |
| `gold/` | 골드 문제 풀이 |

---

## 📄 파일명 규칙

```plaintext
S10828_스택.java
```

| 접두어 | 난이도 |
|--------|--------|
| `B` | Bronze |
| `S` | Silver |
| `G` | Gold |

> 문제 번호 뒤에는 언더바(_)로 구분해 **문제 이름을 붙여 가독성을 높였습니다.**
